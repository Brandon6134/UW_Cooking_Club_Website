# UW_Cooking_Club_Website
 UW Cooking Club website to display recipes, events, team, etc.

## Technologies

**Backend**  
- Django 4+  
- django-cors-headers  

**Frontend**  
- Vue 3  
- Vite (build tool)  
- Tailwind CSS (utility-first CSS framework)

## Getting Started

### Django
```
cd UWCookingWebsite
python manage.py runserver
```

### Vue / Front End
New Terminal
```
cd my-vite-app
npm install
npm run dev
```
